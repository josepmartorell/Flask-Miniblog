Patch Deployment

Web Scrapers 2.0 In this blog we publish a collection of patches that provide answers and solutions to various problems or questions that arise when developing Python Web Scraping applications using Requests, Beautifulsoup4 or Scrapy sometimes in combination with frameworks such as Selenium or Splash to render JavaScript on websites that make use of AJAX. By clicking on the Documentation & Downloads tab, you will access a page with all the technical documentation related to the published patches to date. From the upper right corner you can register or log in directly if you are already, remember that you must do so if you want to leave your comments. Please, do it if you see that you can contribute something to the solutions and tricks presented here.
